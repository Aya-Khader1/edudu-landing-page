# edudu-landing-page

A modern and clean landing page for a kids' education platform inspired by the Edudu design on Figma.  
This project aims to deliver a friendly, playful, and professional UI for young learners.

##  Features
- Fully responsive landing page  
- Organized structure  
- Clean and modern UI  
- Sections:
  - Hero
  - Features
  - Subjects
  - FAQ
  - Footer

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Responsive Design
